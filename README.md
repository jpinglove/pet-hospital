# Pet Hospital 🏥

宠物医院管理系统

## 项目简介

一个宠物医院管理的前端项目，提供宠物医疗服务管理界面。

## 功能特性

- 宠物信息管理
- 预约挂号系统
- 诊疗记录管理
- 药品库存管理

## 开始使用

1. 克隆项目

```bash
git clone https://github.com/jpinglove/pet-hospital.git
cd pet-hospital
```

2. 启动项目

直接打开 `index.html` 或在本地服务器中运行。

## 技术栈

- HTML5
- CSS3
- JavaScript
